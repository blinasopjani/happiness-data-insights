# World Happiness Analysis

Analyze global happiness trends and the factors influencing happiness across countries using the **World Happiness** dataset.  

## Key Features
- Explore happiness scores by country and region  
- Study relationships with GDP, social support, life expectancy, freedom, generosity, and corruption  
- Visualize trends over time with line plots and heatmaps  
- Predict happiness using Linear Regression  
- Group countries with KMeans clustering
